### `실행 방법`

1. git clone 받기
2. 터미널에 ' npm i ' 명령어 실행
3. 터미널에 ' npm run start ' 명령어 실행
4. 셀렉트 박스에서 난이도 설정 시 지뢰찾기 테이블 생성
5. 지뢰찾기 게임 시작

### `사용한 기술`

React, Material UI , Redux-toolkit

### `사용한 라이브러리`

Material UI , Redux-toolkit , nanoid

### `구현 기능`

1. 첫번째 빈칸을 열었을 경우에는 지뢰가 터지지 않는 기능
2. 난이도 변경
